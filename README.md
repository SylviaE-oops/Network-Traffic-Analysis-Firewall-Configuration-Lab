# Network Security Labs

This repository contains hands-on networking and cybersecurity labs completed using Windows 11, Ubuntu Linux, VirtualBox, and Wireshark.

## Objectives

- Capture and analyze network traffic with Wireshark
- Understand ICMP communication
- Configure and test Linux firewall rules
- Observe encrypted SSH traffic
- Analyze DNS queries and responses
- Examine Remote Desktop Protocol (RDP) traffic

## Lab Environment

### Virtual Machines

| Operating System | Purpose |
|-----------------|---------|
| Windows 11 | Client Machine |
| Ubuntu Linux | Server Machine |

### Networking

| Adapter | Purpose |
|---------|---------|
| Host-Only Adapter | VM-to-VM communication |
| NAT Adapter | Internet access |

### Tools Used

- Wireshark
- Windows PowerShell
- Ubuntu Terminal
- OpenSSH Server
- UFW
- iptables
- VirtualBox
